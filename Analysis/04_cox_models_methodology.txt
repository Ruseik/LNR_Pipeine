Overall survival was analysed using Cox proportional hazards regression models implemented in R with the rms and survival packages, after setting the data distribution object with datadist for model support and prediction. The analysis used a three-layer symmetric model-comparison framework: Layer 1 compared unadjusted models with N-stage alone versus lymph node ratio (LNR), modelled either as a linear term or, when permitted by the analytic tier, with restricted cubic splines (RCS) using three knots; Layer 2 added total lymph node examined (TLE) to compare N-stage + TLE versus LNR + TLE; and Layer 3 fitted the primary fully adjusted multivariable models, forcing entry of age, sex, T-stage group, grade, and lymphovascular invasion (LVI) alongside either N-stage or LNR plus TLE, again using RCS with three knots when allowed, otherwise linear terms. An additional interaction model was prespecified to evaluate effect modification between LNR and TLE adequacy using a multiplicative interaction term (LNR × TLE adequacy). Proportional hazards assumptions were assessed for each fitted model using the Schoenfeld residual-based cox.zph test with a log-time transformation, and any violation was flagged for potential stratification or time-dependent modeling. Model performance and parsimony were compared using the Akaike information criterion (AIC), Bayesian information criterion (BIC), and partial log-likelihood, and the adjusted association between LNR and mortality was visualized using predicted hazard ratios from the final multivariable model; a forest plot of hazard ratios with 95% confidence intervals was also generated from the corresponding Cox model.
