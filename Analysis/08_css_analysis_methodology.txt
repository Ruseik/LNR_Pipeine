Cancer-specific survival (CSS) was analyzed as a competing-risks outcome, with non-cancer death treated as the competing event and alive/censored patients coded as censored observations. CSS analyses were performed only when cause-of-death information met the prespecified completeness threshold of at least 85%; otherwise, only descriptive results were reported. Cumulative incidence functions (CIFs) for cancer-specific death were estimated using the nonparametric cumulative incidence approach implemented in the Fine and Gray competing-risks framework, and group differences were assessed with Gray’s test. Subdistribution hazard ratios (sHRs) and 95% confidence intervals were then estimated using the Fine-Gray regression model. Separate Fine-Gray models were fitted for N stage, lymph node ratio (LNR) as a continuous variable, and LNR categorized into tiers. When the analytic tier permitted multivariable modeling, fully adjusted Fine-Gray models were additionally fitted including N stage or LNR together with the covariates total number of examined lymph nodes, age, sex, T-stage group, tumor grade, and lymphovascular invasion. All analyses were conducted using the cmprsk package in R.
